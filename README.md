This project is for research at the University of Victoria.

- Modes: RSVP and Continuous
- Continuous supports viewport steps: letter/word variants + sentence + page
- Controls: viewport step, advance step (RSVP), autoplay speed, typography
- Optional Mouse Y rate control (toggleable min/max CPS mapping)
- Settings: import/export condition JSON from the in-app modal
- Default text: edit `public/default-text.txt` (loaded on startup)
